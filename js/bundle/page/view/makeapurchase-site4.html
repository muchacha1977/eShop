Программа 4. «Чаще покупал за месяц»

 

Период активности бонусной программы – месяц. В течение этого периода QWERAS накапливает количество заказанных уникальных товаров в разрезе покупателей. При этом учитываются только те покупки, которые не получили статус «Отменен». По окончании этого месяца список покупателей, которые совершили заказ покупок, ранжируется от большего количества заказанных уникальных товаров к меньшему количеству. В результате получается следующая последовательность (пример условный, цифры приводятся только для объяснения принципа работы бонусной программы):

Программа 2. «Чаще покупал за день»

 

Период активности бонусной программы – сутки. В течение этого периода QWERAS накапливает количество заказанных уникальных товаров в разрезе покупателей. При этом учитываются только те покупки, которые не получили статус «Отменен». По окончании этих суток список покупателей, которые совершили заказ покупок, ранжируется от большего количества заказанных уникальных товаров к меньшему количеству. В результате получается следующая последовательность (пример условный, цифры приводятся только для объяснения принципа работы бонусной программы):
